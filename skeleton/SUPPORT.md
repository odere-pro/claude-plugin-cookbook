Need help with `example-plugin`? Here's where to look.

## Self-service first

- **Using it** — see [`README.md`](README.md) for what each component does and how to install.
- **Is something structurally broken?** — run `/doctor` in a session for a quick health check.

## Questions and bugs

- **Bugs** — open a GitHub issue. Include the command you ran, your Claude Code version, and the
  relevant `/plugin` or `/doctor` output.
- **Ideas / feature requests** — open a GitHub issue describing the problem and the proposed behaviour.
- **Security issues** — do **not** open a public issue; see [`SECURITY.md`](SECURITY.md).

## Contributing

If you want to fix or extend something yourself, start with [`CONTRIBUTING.md`](CONTRIBUTING.md).
